## 合并两个有序链表

> 将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 
>
>  
>
> 示例：
>
> 输入：1->2->4, 1->3->4
> 输出：1->1->2->3->4->4
>
> 链接：https://leetcode-cn.com/problems/merge-two-sorted-lists



```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

# 递归
def merge_two_lists(self, l1: ListNode, l2: ListNode) -> ListNode:
    if l1 and l2:
        if l1.val > l2.val:
            l1, l2 = l2, l1
        l1.next = self.mergeTwoLists(l1.next, l2)
        
    return l1 or l2
    
# 迭代
def merge_two_lists(self, l1: ListNode, l2: ListNode) -> ListNode:
    pre = ListNode(0)
    cur = pre

    while l1 and l2:
        if l1.val > l2.val:
            l1, l2 = l2, l1
        cur.next = l1
        l1 = l1.next
        cur = cur.next  
    cur.next = l1 or l2
    
    return pre.next
```
